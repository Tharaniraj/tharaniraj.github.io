Featured Projects

Production ⭐ Featured
SDN Data Centre Fabric
Modernised a legacy STP-based DC into a scalable multi-tenant VXLAN/EVPN CLOS fabric across HQ & DR sites
Problem

Legacy STP/MSTP fabric caused slow convergence (30–60 s failover), broadcast storms under load, and couldn't support multi-tenancy for growing banking workloads across HQ and DR.

Solution

Designed a 2-tier Spine-Leaf CLOS architecture on Cisco Nexus NX-OS. Deployed BGP EVPN as control plane and VXLAN as data plane with L2/L3 VNIs, distributed anycast gateways, and ARP suppression for tenant isolation.

Cisco Nexus NX-OS VXLAN / EVPN BGP EVPN Spine-Leaf CLOS Anycast Gateway L3 VNI
✓ Sub-second failover ✓ 99.99% uptime ✓ Zero broadcast storms ✓ Full multi-tenancy
Production ⭐ Featured
Hybrid DC-Cloud Connectivity
Private dedicated multi-cloud connectivity for a Tier-1 bank via AWS & OCI with automatic BGP failover
Problem

Banking applications hosted on AWS and Oracle Cloud required private, low-latency connectivity from on-premise DC. Public internet paths failed PCI DSS compliance and SLA requirements.

Solution

Provisioned AWS Direct Connect with Transit Gateway for AWS workloads, OCI FastConnect for Oracle Banking Core, and Azure ExpressRoute as tertiary path. Configured BGP route policies with MED/AS-path for automatic failover between providers.

AWS Direct Connect AWS Transit Gateway OCI FastConnect BGP Route Policy Azure ExpressRoute
✓ Sub-10ms latency SLA ✓ Zero-downtime migration ✓ PCI DSS compliant ✓ Auto BGP failover
